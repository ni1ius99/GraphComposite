GraphComposite

Programmatic graph building and transformations between popular graph markup formats.

Download this project as a .zip file Download this project as a tar.gz file

Intro

Written in C#. Allows for building directed acyclic graphs (DAGs), and hierarchical composite graphs (inspired by dot - see http://www.graphviz.org/).

This software is released under the Gnu General Public License.

Major Features

  • Programmatic building of in-memory DAGs
  • Convert between in-memory DAG and C# XElements

Planned Updates

  • Incorporate NDoc documentation
  • Convert between in-memory DAG and JSON
  • Convert pivot table data to a DAG
  • Convert between in-memory DAG and SVG
  • Configure non-hierarchical (flat-only) build mode.
  • Configure a tree-only build mode.
  • AddSubgraph method.
  • Merge graphs using two root nodes.
